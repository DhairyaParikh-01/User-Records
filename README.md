
# User Records 

CRUD application for displaying, editing, updating & removing a user record. 


## Features

- CRUD oprations
- Light weighted
- Fully Responsive 
- ReGex Validation


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, Bootstrap
